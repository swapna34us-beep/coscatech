    Human Contribution Index

    HCI Formula (Paper §6)
    HCI = (H_F × H_D × H_R)^(1/3) — geometric mean, non-compensable.
    H_F = Frame Contribution: authorship + documentation + challenge
    H_D = Decision Contribution: structural position in decision loop
    H_R = Residue Contribution: surfacing + authorization + timeliness

    HCI = 0 when any sub-index is 0. A system with strong frame authority but no residue surfacing is not "partially governed."
