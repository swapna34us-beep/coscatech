What Is This Experiment?
The question, the method, and the governance document
Research Question: Does writing a formal governance document (GRD) before building an AI system produce measurably different — and measurably better — outcomes than standard ML practice?
Experiment Design
Two groups built the same AI system (supplier recommendation for indirect procurement) from the same 10,000-transaction synthetic dataset.

Control Group: Standard ML best practice. No governance document. Single target: 15% cost reduction. They received the visible dataset (16 columns) and built a model.

Treatment Group: Before touching data, they wrote a complete Governance Reference Document (GRD) specifying: one goal with 6 criteria, 5 rules, 5 thresholds, 5 residue domains, 3 proxy exclusions, and a deployment level. Then they built only what the GRD authorized.

The test: Score both systems on 19 metrics across 3 layers (Input, Process, Output), evaluate 5 claims, and diagnose threshold miscalibrations using Laplace-domain and Lyapunov switched-system analysis.
The GRD — Governance Rule Document (GRT-G-002)
The 5 Claims Being Tested
C1: Governance surfaces problems earlier (proxy detection, frame-mismatch flagging)
C2: Governance changes the design (goal maturity, rules, boundary enforcement, authority routing)
C3: Governance changes the deployment decision (explicit level declaration vs default)
C4: Framework metrics produce real signal (aggregate across all 19 metrics)
C5: The cost of governance is worth it (authoring cost vs measurable value in bias, transparency, auditability)

What the Human Knows (Hidden Fields)
These 5 fields represent the buyer's situated knowledge — information the model structurally cannot access:

diversity_certified — the protected attribute. The model must not use this, but proxy features can leak it.
relationship_years — how long the buyer has worked with this supplier.
buyer_trust_score — the buyer's subjective trust in the supplier.
disruption_resilience — how well the supplier handles supply chain disruptions.
strategic_alignment — whether the supplier is strategically important beyond cost.

The GRT framework calls this gap residue. The HCI score measures the value of that residue.

Stage 3: Threshold Calibration
Testing whether governance thresholds fire at the right rate
Test: For each threshold, fire it against all 10,000 transactions. Compare the fire rate against the GRD-declared target band. Diagnose miscalibrations using two control-theoretic frameworks:

Laplace (§4.4): Thresholds act as controller gains. Under-firing = over-damped (sluggish). Over-firing = marginally stable (oscillation risk).
Lyapunov (§5.3): Checks whether the switched system can cross regime boundaries stably. Frequent switching = check dwell-time condition.
4
Stage 4: Human Contribution Index
Measuring where human judgment enters the decision loop
Test: Compute HCI for both groups. HCI = (H_F × H_D × H_R)^(1/3). Geometric mean is non-compensable — if any sub-index is zero, HCI is zero. A system with strong frame authority but no residue surfacing is not "partially governed."

H_F (Frame): Did a human set the goal, document it, and retain challenge authority?
H_D (Decision): Where in the decision loop does the human sit? (pre-execution = 1.0, none = 0.0)
H_R (Residue): Is the human informed of what was dropped in translation from intent to action?

Stage 6: Claim Verdicts
What the experiment proved — and what it didn't
Verdict rules: CONFIRMED = all primary metrics pass. PARTIALLY CONFIRMED = directionally favorable but not strongest outcome. PENDING = requires post-deployment data.
What the Experiment Does NOT Show
Durability: M10–M13 sustainability metrics require post-deployment data. The framework pays for its authoring cost in a single run; the multi-run case is untested.

Portability: Tested in indirect procurement only. Portability to clinical, hiring, or financial domains where Level 4/5 decisions are more likely is future work.

Proxy strength: The dataset's proxy correlations are weaker than declared (employee_count r=-0.503 vs target -0.58, hq_region r=0.179 vs target +0.47). The framework conclusion holds but the quantitative effect is understated.
